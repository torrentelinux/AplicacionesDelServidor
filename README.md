# AplicacionesDelServidor
"AplicacionesDelServidor", versión 1.0, para Windows

Utilitario Freeware para Windows 32/64 bits.
Orientado para la administración de servidores y la gestión de centros de datos.

Instale la aplicación .exe junto con wgetver.ps1, sus librerías .dll y .bpl en
el directorio/carpeta de su elección.
Ejecute 'AplicacionesDelServidor.exe' haciendo clic sobre el mismo.

Para desinstalar la aplicación .exe junto con .ps1 y sus librerías .dll y .bpl
borre el directorio/carpeta por completo que contiene a los archivos antes mencionados.
