# AplicacionesDelServidor
"AplicacionesDelServidor", versión 2.0, para Windows<br>

Utilitario Freeware de 32 bits para Windows 32/64 bits.<br>
Orientado para la administración de servidores y la gestión de centros de datos.<br>
Lo hace ideal para correr en Windows Server Core en todas sus versiones.<br>

Para instalar la aplicación .exe primero descomprima el fichero<br>
'AplicacionesDelServidor-v2_0-binarios-win32.7z' en el directorio/carpeta<br>
de su elección.<br>
Ejecute 'ads.exe' haciendo clic sobre el mismo.<br>

Para desinstalar la aplicación y todos sus ficheros borre el directorio/carpeta<br>
por completo que contiene a los archivos abajo mencionados.<br>

Los ficheros son:<br>
    <·> AplicacionesDelServidor-Win32bit_b.ico<br>
    <·> ads.exe<br>
    <·> leame.txt<br>
    <·> wgetver.ps1<br>
    <·> borlndmm.dll	· Librería externa de Embarcadero Technologies, Inc.<br>
    <·> cc32280mt.dll	· Librería externa de Embarcadero Technologies, Inc.<br>
    <·> rtl280.bpl		· Librería externa de Embarcadero Technologies, Inc.<br>

Desde la línea de comandos del sistema escriba:<br>
	--> ads /?<br>
	--> ads /i<br>

y mostrará los detalles de producción del mencionado software.<br>
