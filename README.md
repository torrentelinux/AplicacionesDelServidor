# AplicacionesDelServidor
"AplicacionesDelServidor", versión 3.0, para Windows<br>

Utilitario Freeware de 64 bits para Windows 64 bits.<br>
Orientado para la administración de servidores y la gestión de centros de datos.<br>
Lo hace ideal para correr en Windows Server Core en todas sus versiones.<br>

Archivo instalador<br>
Para instalar la aplicación .exe primero ejecute el instalador<br>
'AplicacionesDelServidor-Instalador-3.0.exe' y siga los pasos<br>
del asistente de la instalación.<br>
Cuando termine el proceso de instalación ejecute el acceso directo<br>
que se encuentra en el Escritorio 'AplicacionesDelServidor.lnk' haciendo doble clic sobre el mismo.<br>
Para desinstalar la aplicación y todos sus ficheros vaya a 'Control Panel\Programs\Programs and Features'<br>
y haga doble clic sobre 'AplicacionesDelServidor', publicador 'Torrentarium'.<br>

Archivo comprimido<br>
Para instalar la aplicación .exe primero descomprima el fichero<br>
'AplicacionesDelServidor-v3_0-binarios-win64.7z' en el directorio/carpeta<br>
de su elección.<br>
Ejecute 'ads.exe' haciendo clic sobre el mismo.<br>
Para desinstalar la aplicación y todos sus ficheros borre el directorio/carpeta<br>
por completo que contiene a los archivos abajo mencionados.<br>

Los ficheros y directorios son:<br>
    <·> [CapturasDePantalla]	· Directorio que contiene archivos imágenes del tipo PNG.<br>
    <·> [Dependencias]		· Directorio que contiene información técnica acerca del producto.<br>
    <·> [LICENCIA]			· Directorio que contiene la licencia del producto (español/inglés).<br>
    <·> AplicacionesDelServidor-Win32bit_b.ico<br>
    <·> ads.exe<br>
    <·> ads-gui.exe<br>
    <·> ads-tui.exe<br>
    <·> leame.txt<br>
    <·> wgetver.ps1<br>
    <·> borlndmm.dll		· Librería externa de Embarcadero Technologies, Inc.<br>
    <·> cc64280mt.dll		· Librería externa de Embarcadero Technologies, Inc.<br>
    <·> rtl280.bpl   		· Librería externa de Embarcadero Technologies, Inc.<br>
    <·> vcl280.bpl   		· Librería externa de Embarcadero Technologies, Inc.<br>
    <·> vclimg280.bpl		· Librería externa de Embarcadero Technologies, Inc.<br>

Desde la línea de comandos del sistema escriba:<br>
	ads /?<br>
	ads /i<br>

y mostrará los detalles de producción del mencionado software.<br>

Atención<br>
Tenga muy presente que este producto software está destinado para el administrador del sistema operativo,<br>
cuenta 'Administrador'/'Administrator'.<br>
